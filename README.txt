Добрый день, в тестовом задании я описал примерную структуру проекта, которую использую при разработке web приложений. Приложение разделил на две части 
backend – это Web Api на .net core 6. Front - части использовал Asp.Net core(MVC) и немного html, css. Так как я являюсь backend разработчиком, особое 
внимание я уделил именно backend части. При разработке я использую трехуровневую архитектуру построения приложения (уровень данных, уровень бизнес логики, 
сами сервисы(контроллеры)). СУБД использовал postgre и ORM Entity Framework модели прокидывал в БД через миграции. Также добавил возможность логгирования в БД, 
использовал библиотеку NLog. Для мапинга Model в DTO использовал библиотеку AutoMapper. В качестве репозитория взял ранее разработанный мной класс, 
так же использовал паттерн единственного репозитория UnitOfWork. Для документации и тестирования Api использовал Swagger. Сам сервис локально развернул на сервере IIS.
